# LearnTrack - Setup Instructions

## 1. Java Installation

This project is built using **Java Development Kit (JDK) 21**.

### Steps to install JDK:
1. Download JDK from:
   https://www.oracle.com/java/technologies/javase/jdk21-archive-downloads.html  
   OR  
   https://adoptium.net/

2. Install the JDK by following the installer steps.

3. Verify installation using the terminal or command prompt:

java -version
javac -version


If Java is installed correctly, the version details will be displayed.

---

## 2. IDE Setup

This project is developed using **IntelliJ IDEA**.

Steps:
1. Open IntelliJ IDEA.
2. Click on **New Project**.
3. Select **Java**.
4. Choose the installed JDK.
5. Name the project as `LearnTrack`.
6. Click **Finish**.

---

## 3. Running the Application

1. Open the file `Main.java`.
2. Right-click inside the file.
3. Click **Run 'Main'**.
4. The output will appear in the console.
